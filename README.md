# nfc tool share key
##  nfc门禁破解共享密钥，免vip使用联动密钥破解加密门禁卡

本项目将不定期更新密钥~~~~


## 使用方式

### 方式一：使用官方 NFC Tool APP
1.  下载密钥txt文件到手机目录
2.  安装nfc_tool.apk
3.  打开app -> 一键读写-> 点击使用联动密钥按钮-> 点击加载-> 选择密钥txt文件-> 点击确定-> 开始读卡-> 等待破解完成

其他模拟门禁过程参考官方文档

#### 下载文件建议每8000个密钥一个文件，避免文件过大，加载密钥导致软件崩溃。
#### 分割文件代码可以参考 spliteKey.js,  在node.js 环境下使用

### 方式二：使用本项目的 Android 扫描 APP（推荐）

本项目提供了一个独立的 Android 应用，内置密钥库，无需下载额外文件，支持自动扫描 M1 卡密钥。

#### 功能特点
- 内置密钥库，开箱即用
- 自动扫描 Mifare Classic 1K/4K 卡片密钥
- 支持断点续扫（卡片离开后可继续扫描）
- 扫描记录自动保存
- 支持多种格式导出（JSON/文本/MFD）
- 实时显示扫描进度和速度

#### 安装步骤
1. 打开 [Releases](https://github.com/hugh999999/nfc_tool_share_key/releases/latest) 下载最新 APK（`Assets` 区）
2. 安装到支持 NFC 的 Android 手机（需 Android 6.0+）
3. 确保 NFC 功能已开启

#### 使用方法
1. **开始扫描**
   - 打开 APP，点击「开始扫描」按钮
   - 将 M1 卡片靠近手机背面 NFC 区域
   - APP 会自动检测卡片并开始扫描密钥

2. **扫描过程**
   - 实时显示已找到的密钥和扫描进度
   - 显示当前扫描速度（次/秒）
   - 如卡片离开，会自动保存进度

3. **断点续扫**
   - 如果扫描过程中卡片离开，进度会自动保存
   - 重新靠近卡片后，会从上次位置继续扫描

4. **查看记录**
   - 点击右上角「记录」按钮查看所有扫描历史
   - 显示每张卡的 UID、扫描时间、完成状态

5. **导出记录**
   - 在记录页面，点击单条记录的「导出」按钮
   - 选择导出格式：
     - **JSON 格式**：完整的结构化数据，适合程序处理
     - **文本格式**：易读的文本报告
     - **MFD 格式**：标准 Mifare Classic Dump 格式，可用于复制卡片

6. **批量导出**
   - 在记录页面点击「导出全部」
   - 导出所有卡片记录为 JSON 文件

#### 导出格式说明

**JSON 格式示例：**
```json

  "uid": "A1B2C3D4",
  "scanTime": 1712123456789,
  "cardType": "Mifare Classic",

    "1": { "keyA": "FFFFFFFFFFFF", "keyB": null }
  }
}
```
